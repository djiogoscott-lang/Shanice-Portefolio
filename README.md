# Portfolio — Mago Kamdem Schanice

Portfolio personnel **One-Page** d'une conceptrice d'interfaces et développeuse web, dédiée à un design éthique, accessible et centré sur l'humain.

## Aperçu

Un portfolio statique, élégant et raffiné, construit avec un soin particulier pour :

- Une esthétique éditoriale (typographie *Cormorant Garamond* + *Inter*)
- Des micro-interactions fluides (hover, fade-in au scroll)
- Une accessibilité respectueuse (`prefers-reduced-motion`, ARIA labels)
- Un design 100 % responsive

## Sections

- **Hero** — Présentation et invitation
- **À propos** — Vision et valeurs
- **Projets** — Application Calculatrice & Site Météo Dynamique
- **Philosophie** — Citation éditoriale
- **Contact** — Liens `mailto:` et `tel:` interactifs

## Technologies

- **HTML5** sémantique
- **CSS3** moderne (Flexbox, Grid, Custom Properties, `backdrop-filter`)
- **JavaScript Vanilla** (menu burger, `IntersectionObserver`, glow cursor)

## Lancer en local

Aucune dépendance, aucun build. Ouvre simplement `index.html` dans un navigateur, ou sers le dossier avec un serveur statique :

```bash
npx serve .
```

## Déploiement

Déployé sur **Vercel** — déploiement automatique à chaque push.
